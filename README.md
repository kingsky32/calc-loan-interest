# 원리금 균등상환, 원금 균등상환 계산 js

## 사용법

```js
calcLoanInterest(
  0, // method: 상환방법 (0: 원리금 균등 상환, 1: 원금 균등 상환);
  300000000, // loanMoney: 원금
  0.03, // rates: 이자율
  24, // loanDate: 납입기간(월)
  0, // period: 거치기간(월)
);
```
